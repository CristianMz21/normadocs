# ICONTEC Configuration

## Default Configuration

```yaml
fonts:
  body:
    name: "Arial"
    size: 12
margins:
  unit: cm
  top: 3.0
  bottom: 3.0
  left: 3.0
  right: 4.0
spacing:
  line: 1.5
```

## Formatting Rules

- Font: Arial 12pt throughout
- Line spacing: 1.5
- Margins: 3cm (top, bottom, left), 4cm (right)
- Cover page: Specific structure required
